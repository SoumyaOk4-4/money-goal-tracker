# 💸 Moni Goal Tracker

A sleek and modern money tracking web app built with **React + Tailwind CSS** to help manage savings, spending, and financial goals in a simple way.

Built because I spend a lot sometimes and wanted a clean app to keep track of:
- 💰 Savings
- 💸 Spendings
- 🏦 Online Money
- 👛 Cash
- 🤝 Lent Money
- 📈 Goal Progress

---

# 🌐 Live Demo

👉 https://moni-goal.netlify.app/

---

# ✨ Features

## 📊 Smart Tracking
- Add money transactions instantly
- Use negative values for spendings
- Track multiple categories:
  - Cash
  - Online
  - Lent
  - Others

---

## 🎯 Goal System
Set a money goal and track your progress visually with:
- Animated progress bar
- Real-time percentage tracking
- Clean financial dashboard

---

## 📈 Analytics
- Interactive pie chart
- Money distribution visualization
- Clean UI for better understanding

---

## 💾 Backup System
### 📤 Export Backup
Export all your saved data into a `.json` file.

Useful for:
- Browser switching
- System reinstall
- Device transfer
- Personal backup

---

### 📥 Import Backup
Import previously exported backup files and restore:
- Transactions
- Goal amount

---

## 🔄 Reset App
Reset everything instantly with a confirmation popup.

Removes:
- Transactions
- Goal data
- Stored browser data

---

# 🧠 How It Works

The app uses your browser's **Local Storage**.

That means:
- ✅ No login required
- ✅ No backend/server
- ✅ Works offline
- ✅ Data stays on your device

---

# ➕ Adding Transactions

## Add Savings

```txt
5000
```

Adds ₹5000.

---

## Track Spending

```txt
-500
```

Subtracts ₹500 from your total.

Example:

```txt
-299
```

Means you spent ₹299.
